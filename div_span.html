Tags <DIV>

Usada para reservar um espaço no código para posteriormente usar CSS ou JAVASCRIPT para manipulá-lo.

```

Uma DIV é do tipo Dispray Block, ela ocupa todo o espaço na horizontal de onde ela está inserida

```

TAG <SPAN>

Diferente do modelo da div, a SPAN não ocupa toda a horizontal, apenas o trecho que foi envelopado.

É usada para mudar apenas um trecho da frase, ou uma letra da palavra e etc...O que é impossível fazer smomente com o html
