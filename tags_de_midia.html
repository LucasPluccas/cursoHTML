O que são as tags de mídia?

Tags usadas especificamente para anexar mídias no seu código HTML.

Tag <img>

Usada para anexar imagens, gifs e etc...

Tag img

atributo Title e alt

atributo Title serve para nomear a imagem anexada

Atributo alt serve para fazer uma descrição da imagem, para auxilio para deficientes, e leitura de robo

Tag Audio

Usada para anexar exclusivamente audio


Tag <vídeo>

Usada para anexar vídeos
